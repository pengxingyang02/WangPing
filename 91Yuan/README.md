项目介绍
 
一个纯前端整蛊恶作剧页面，无任何真实色情内容、无恶意代码、无追踪、无广告。
页面伪装成常见的“成人网站”风格，当朋友点击进入并试图播放视频时，
会自动跳转到/播放：
 
- 魔性骷髅跳舞 1 小时纯享版
 
仅用于朋友间玩笑、整活、整蛊，禁止用于任何恶意用途。
 
功能
 
- 仿钓鱼网站视觉风格（仅界面，无不良内容）
- 点击播放 → 骷髅跳舞1小时
- 纯静态 HTML + CSS + JS，可直接部署
- 无后端、无采集、无隐私获取
 
使用方法
 
1. 克隆或下载本项目
2. 直接打开  index.html  即可预览
3. 可部署到 GitHub Pages 需要自行替换仓库名称和用户名
4. 把链接发给朋友，等待爆笑（或绝交）现场
 
部署示例（GitHub Pages）
 
1. Fork 本项目
2. 进入 Settings → Pages
3. Source 选择  main  +  /root 
4. 等待几分钟即可获得公开整蛊链接
